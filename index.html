ConsoleMessage NodeJS Library


ConsoleMessage is a Javascript library to write colored logs on console, it'll helpful to identify type of log message by looking at color. ConsoleMessage as function called log which takes two parameters that are message and the flag. message is the log message which user wants to printed on console. flag is the indication for type of log message. List of flags are SUCCESS, DEBUG, INFO, WARNING, ERROR, CRITICAL. You can find more about this in npm.

  npm install -g consolemessage


//logger.js

var console = require('consolemessage');

console.log('info message', console.info);
console.log('success message', console.success);
console.log('warning message', console.warning);
console.log('debug message', console.dubug);
console.log('error message', console.error);
console.log('crtitical message', console.critical);


//output

success message

debug message

info message

warning message

error message

critical message
